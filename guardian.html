Guardian of the Code

Guardian of the Code is a visual novel designed to teach about internet safety and general cyber security concepts.
This game was created using Ren'Py as a final college project at the University of Washington Bothell. I designed the majority of the game.

About the Game

Guardian of the Code is fully playable within an internet browser and is best played on a computer.
The player must explore The Digital World as they learn about internet safety and solve puzzles to test their knowledge. The ending is determined by how safe the player is when navigating through the world. The game is designed to inform and test on the best practices of internet safety such as identifying urls, password creation, avoiding phishing scams, and more.

Game length: About 30 minutes.
