Guardian of the Code

Guardian of the Code is a visual novel designed to teach about internet safety and general cyber security concepts.
This game was created as a final college project at the University of Washington Bothell. I designed the majority of the game.

About the Game

Guardian of the Code is fully playable within an internet browser and is best played on a computer. It is designed to inform and test on the best practices of internet safety such as identifying urls, password creation, avoiding phishing scams, and more. It can take 30-45 minutes to go through the game but there are mulitple endings that depend on how well you learn the concepts.
